**v1.2.0** (2019-08-31)
- 支持关闭/打开数据上报
- 支持暂停/继续数据上报
- 支持轻量级实例
- 修复2019.2.1f 版本设置网络类型失败
- 修复timeEvent 不准确问题
- Android/iOS SDK 升级为2.1.0

**v1.1.0** (2019-08-09)
- 支持获取设备ID
- 支持动态公共属性
- 支持安装事件采集
- Android SDK 默认升级为2.0.2
- iOS SDK 升级为2.0.1
- 支持上报缓存选项，用户可以选择主动调用StartTrack() 去开始上报

**v1.0.0** (2019-06-20)
- 支持访客ID 和用户账号的设置
- 支持事件和用户属性上报
- 支持 ta_app_start 和 ta_app_end 事件的自动上报
- 支持公共属性接口
- 支持 timeEvent 接口
- 支持多项目上报
